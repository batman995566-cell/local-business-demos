# Elegant Tailor Demo Site

A polished, calm, and upscale demo website for a luxury tailor shop, inspired by high-end menswear brands.

## Features

- Clean, elegant layout with a focus on typography and color scheme
- Responsive design that works on desktop and mobile
- Smooth scrolling navigation
- Mobile-friendly hamburger menu

## Color Scheme

- **Primary Colors**: Deep Navy (#001122), Cream/Off White (#F5F5DC), Soft Charcoal (#36454F)
- **Accent Colors**: Gold (#D4AF37), Brass (#B5A642), Subtle Beige (#F5DEB3)

## Typography

- Headings: Playfair Display (serif, elegant and formal)
- Body: Montserrat (sans-serif, clean and modern)

## Usage

1. Open `index.html` in your web browser to view the site.
2. For a better experience, serve the files using a local web server to avoid CORS issues with fonts.

### Running Locally

To run the site with a local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Files

- `index.html`: Main HTML structure
- `styles.css`: CSS styles with custom properties for colors
- `script.js`: JavaScript for smooth scrolling and mobile menu
- `README.md`: This file

## Customization

- Edit colors in the `:root` section of `styles.css`
- Modify content in `index.html`
- Add images by replacing placeholder backgrounds or adding `<img>` tags

## Browser Support

Works in all modern browsers. Tested on Chrome, Firefox, Safari, and Edge.
